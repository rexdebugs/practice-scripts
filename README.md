## Day 23 — Running programs from WSL

### How to run a shell script
Create a file like `hello.sh`, put shell commands in it, and run it with `bash hello.sh`.

### How to run a Python file from terminal
Create a file like `hello.py`, put Python code in it, and run it with `python3 hello.py`.
